Data tools

This module provides functionality for retrieving links to images and masks stored in specified directories.

data_tools.retrieveLinksForFolder(config: Dict[str, Any]) → Tuple[List[str], List[str]][source]

This function retrieves the links for all images and their corresponding masks in a specified folder.

Parameters:

config (Dict[str, Any]) – Configuration dictionary with keys “IMAGE_FOLDER” and “NUMBER_OF_IMAGES”

Returns:

Lists of image and mask filepaths

Return type:

Tuple[List[str], List[str]]

data_tools.retrieveLinksForOneImageInEachFolder(config: Dict[str, Any]) → Tuple[List[str], List[str]][source]

This function retrieves the links for one image and its corresponding mask from each subfolder within a specified directory.

Parameters:

config (Dict[str, Any]) – Configuration dictionary with key “BASE_DIR”

Returns:

Lists of image and mask filepaths

Return type:

Tuple[List[str], List[str]]
